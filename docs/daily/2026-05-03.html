MCP vs Agent Skills: Architectural Decision Framework
MCP is a JSON-RPC client-server protocol connecting N agents to M backends via a separate runtime process — suited for live system/data integration. The key decision axis: reach external systems → MCP; encode reusable procedural know-how → Skills.
